WP eBay Daily Deals - allows you to easily insert eBay Daily Deal listings into your WP sidebar, and earn commission from sales.

== Description ==

Allows you to insert a listing of the eBay Daily Deals in your sidebar.
If you include an eBay Partner Network campaign ID, you can earn money from traffic you drive to ebay!

A few of the features:

*   Choose how many deals to show
*   Only show a specific deal catagory if you like
*   Allows you to use the eBay Partner Network
*   Choose the logo width to fit your sidebar
*   Hides the affiliate link to ebay

== Installation ==

WP eBay Daily Deals is installed like any other Wordpress plugin.

1) Download the latest package from the downloads page (http://wordpress.org/extend/plugins/wp-ebay-daily-deals/).

2) Upload the whole directory into your wordpress plugins directory (usually, yoursite/wp-content/plugins). The plugin must reside in a sub-folder called "wp-ebay-daily-deals" (lower-case!) as supplied in the zip file, else it WON’T WORK

3) Go into the Wordpress admin panel, find Plugins and WP eBay Daily Deals should be listed. Click Activate to activate!


Using WP eBay Daily Deals

1) Go to your theme widgets and find the WP eBay Daily Deals widget and place it.

2) Configure the settings in the options panel.

Check out the [Demo](http://ljapps.com/wp-ebay-daily-deals-wordpress-plugin/ "Demo") and ask questions in the comments section on that same page.
